HADC_HARDI

PURPOSE

compute the ADC from HADC model for ngdir directions.

SYNOPSIS

function [ADC_HADC_cmpts_alldir,ADC_HADC_allcmpts_alldir,ctime_alldir]= HADC_HARDI(experi_hadc,mymesh,DIFF_cmpts,IC_cmpts,points_gdir,graddir_index,negii)

DESCRIPTION 

 compute the ADC from HADC model for ngdir directions.
 
 Input:
     1. experiment_hadc is a structure with 8 elements:
         ngdir_total 
         gdir       
         sdeltavec   
         bdeltavec    
         seqvec       
         npervec     
         rtol        
         atol               
     2. mymesh is a structure with 10 elements:
         Nnode
         Nele
         Nface
         Pts_cmpt_reorder
         Ele_cmpt_reorder
         Pts_ind
         Pts_boundary_reorder
         Fac_boundary_reorder
         Nboundary
         Ncmpt
     3. DIFF_cmpts
     4. IC_cmpts
     5. points_gdir
     6. graddir_index
     7. negii

 Output:
     1. ADC_HADC_cmpts_alldir
     2. ADC_HADC_allcmpts_alldir
     3. ctime_alldir
