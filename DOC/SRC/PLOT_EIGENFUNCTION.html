PLOT_EIGENFUNCTION

PURPOSE

plot the eigenfunction

SYNOPSIS

function PLOT_EIGENFUNCTION(mymesh,SOL,OUT_cmpts_index,ECS_cmpts_index,IN_cmpts_index,nindex,title_str)

DESCRIPTION 

 plot the eigenfunction

 Input:
     1. mymesh is a structure with 10 elements:
         Nnode
         Nele
         Nface
         Pts_cmpt_reorder
         Ele_cmpt_reorder
         Pts_ind
         Pts_boundary_reorder
         Fac_boundary_reorder
         Nboundary
         Ncmpt
     2. SOL: the set of eigenfunctions
     3. OUT_cmpts_index
     4. ECS_cmpts_index
     5. IN_cmpts_index
     6. nindex: the index of the eigenfunction to be plotted
     7. title_str
     
 Output:
     figures for the eigenfunction
